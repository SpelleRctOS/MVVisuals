MVVisuals

by _SpelleR_

This game was created in text format. All actions what are running in game will be realizing in your head in picture format.
Here works your imagination, to complete the game.

MVVisuals is cross-platform game, you can play on computer and on your mobile phone too.

COMPUTER:
To instal the game on computer you need to install Python 3.8 (https://www.python.org/ftp/python/3.8.1/python-3.8.1.exe)
After installing you need to run MVVisuals.bat
And last thing to do, write this string - python MVVisuals.py

MOBILE PHONE:
To install the game to your phone just simply instal Python3 programm from your mobile phone market.
When it is installed, run Python3 and open there MVVisuals.py file
